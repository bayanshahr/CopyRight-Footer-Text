This plugin automatically appends a custom copyright text to the end of each post.

== Description ==
This plugin allows you to add a custom text (e.g., copyright text) at the end of all posts. The text can be styled using the WordPress editor.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
